PolyBurst

PolyBurst is a 2D Block Puzzle game, built in Unity

This game was originally designed in one week for a Game Jam between me and my friends.

The game itself may not be too complicated on the surface but it sparked my interest in game development and taught me C#.

I made this a cloned game because I wanted to see if I could replicate something into code just from understanding how it plays.

Inside the code I explore the usage of hashmapping, lamba functions, and a unique randomizing system.

The entire game is my code, and there are some new features that include: UI Menu system, Unique pieces(5-square), and a custom randomizer using a tilebag system.

Eternal Labyrinth

Eternal Labyrinth is a metroidvania game, built in Unity

This game was made in one month for the Metroidvania Game Jam on Itchio.

The game uses highly elaborate code for a fluent movement system. The aim was to replicate a similar feel to platformers such as Celeste

On top of the code, I made the art and music entirely on my own, and I am very proud of the results.

The idea of the game is themed heavily around Heaven and Hell, and follows the narrative of a demon child fighting to escape punishment for the sins of his father.
